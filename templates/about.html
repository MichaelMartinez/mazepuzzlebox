Maze Puzzle Box created by...

...Bristol-based inventor, wild-man and general all-round genius creator-of-worlds, Mr Matthew Venn. Matt is a member of Bristol Hackspace, where the boxes are put together. The box parts are cut out by Stuart 'Lunatic' Childs with his Beaver powered Laser.

All code and plans are released under the GPL v3 license. Source code for website and DXF/SVG generation is here: https://github.com/mattvenn/mazepuzzlebox


Big thanks to...

London Hackspace for laser cutting and general help, Johannes for javascript maze designer, Andy for python help, Alice for UI help, kellbot for info on the sdxf library, and the Open source community for supplying such awesome tools: Django, Python and all the rest.
